# VK Multi Audio Upload

Chrome extension that brings back bulk music upload to VKontakte — drag & drop multiple MP3 files simultaneously.

## Installation

1. Download and extract the archive
2. Open `chrome://extensions/`
3. Enable **Developer mode** (toggle in the top-right corner)
4. Click **«Load unpacked»** and select the `vk-multi-upload` folder

## Usage

1. Go to the music page of a community or your profile on vk.com
2. Click the **↑** button next to the standard upload button
3. Drag MP3 files into the window or click **«Select files»**
4. Tracks upload automatically in sequence

## Compatibility

- Chrome, Chromium, Brave, Edge, Yandex Browser
- vk.com (desktop version only)

# VK Multi Audio Upload

Chrome-расширение, которое возвращает массовую загрузку музыки во ВКонтакте — drag & drop нескольких MP3 одновременно.

## Установка

1. Скачай и распакуй архив
2. Открой `chrome://extensions/`
3. Включи **Режим разработчика** (переключатель в правом верхнем углу)
4. Нажми **«Загрузить распакованное расширение»** и выбери папку `vk-multi-upload`

## Использование

1. Перейди на страницу музыки сообщества или своей страницы на vk.com
2. Нажми кнопку **↑** рядом со стандартной кнопкой загрузки
3. Перетащи MP3 файлы в окно или нажми **«Выбрать файлы»**
4. Треки загружаются автоматически по очереди

## Совместимость

- Chrome, Chromium, Brave, Edge, Yandex
- vk.com (только десктопная версия)
