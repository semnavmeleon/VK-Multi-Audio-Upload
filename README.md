# VK Multi Audio Upload

Chrome-расширение для ВКонтакте: массовая загрузка музыки, автоматическое создание плейлистов, поиск дубликатов и скачивание треков.

## Возможности

### Загрузка музыки
- **Массовый drag & drop** — перетаскивай сразу несколько MP3-файлов, они загрузятся по очереди
- **Авто-метаданные** — расширение читает ID3-теги (исполнитель, название, альбом, год) и автоматически заполняет поля при загрузке
- **Парсинг из имени файла** — если тегов нет, метаданные извлекаются из имени файла (формат `Исполнитель - Название.mp3`)
- **Retry при ошибках** — повторная загрузка одним кликом, если трек не загрузился

### Автоплейлист
- После загрузки треков автоматически создаётся плейлист с названием из тегов альбома
- **Обложка** — можно задать свою обложку в настройках, она применится автоматически
- Формат названия: `Альбом (Год) Исполнитель`

### Поиск дубликатов
- Кнопка «Найти дубликаты» в диалоге плейлиста
- Сканирует только треки внутри открытого плейлиста
- Сравнивает по названию и исполнителю
- Удаление дубликатов прямо из диалога

### Скачивание плейлистов
- Кнопка **«Скачать плейлист»** на страницах с музыкой
- Скачивает все треки плейлиста через HLS (поддержка AES-128 шифрования)
- Прогресс-бар и возможность отмены

### Скачивание отдельных треков
- Кнопка загрузки появляется при наведении на любой трек
- Работает на странице «Моя музыка» и на страницах плейлистов
- Получение URL через встроенный плеер VK — обходит обфускацию ссылок

## Установка

1. Скачай и распакуй архив
2. Открой `chrome://extensions/`
3. Включи **Режим разработчика** (переключатель в правом верхнем углу)
4. Нажми **«Загрузить распакованное расширение»** и выбери папку `vk-multi-upload`

## Использование

### Загрузка треков
1. Перейди на страницу музыки сообщества или своей страницы на vk.com
2. Нажми кнопку **↑** рядом со стандартной кнопкой загрузки VK
3. Перетащи MP3-файлы в окно или нажми **«Выбрать файлы»**
4. Треки загружаются автоматически по очереди

### Скачивание плейлиста
1. Открой страницу плейлиста или раздел музыки
2. Нажми кнопку **«Скачать плейлист»**
3. Дождись сбора треков и подтверди скачивание

### Скачивание одного трека
1. Наведи на любой трек в списке
2. Нажми иконку **↓** среди кнопок действий

### Настройки
- Открываются через иконку шестерёнки в панели загрузки
- **Автоплейлист** — автоматически создавать плейлист после загрузки
- **Обложка** — задать изображение для обложки плейлиста
- **Авто-мета** — заполнять метаданные из ID3-тегов

## Совместимость

- Chrome, Chromium, Brave, Edge, Yandex Browser
- vk.com (только десктопная версия)
- Manifest V3

## Структура

```
├── manifest.json        — конфигурация расширения
├── background.js        — service worker (обработка chrome.downloads)
├── injector_early.js    — ранняя инъекция скрипта в контекст страницы
├── injected.js          — скрипт в контексте страницы (доступ к VK API, React fiber)
├── content.js           — основной контент-скрипт (UI, логика загрузки/скачивания)
└── style.css            — стили расширения
```
