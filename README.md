# VK Multi Audio Upload

Расширение для Chromium-браузеров. Расширяет работу с музыкой на vk.com: массовая загрузка, проверка наличия треков на странице, поиск дубликатов, скачивание и удобная настройка раскладки.

## Возможности

### Загрузка

- Перетаскивание группы MP3 в диалог «Добавить аудиозапись» — файлы выстраиваются в очередь и загружаются по одному.
- Состояние каждого трека отображается отдельно, упавшие можно перезапустить или скопировать их имена в буфер.
- Очередь ставится на паузу, отдельные файлы — отменяются.
- Отсутствующие исполнитель и название можно автоматически взять из имени файла по шаблону `Исполнитель - Название.mp3`.

### Авто-плейлист

После окончания очереди расширение само открывает диалог создания плейлиста, заполняет название/описание из тегов и ставит обложку. Обложка — либо выбранная в настройках картинка 1000×1000, либо встроенная APIC из первого трека. На обложку накладывается водяной знак «перезалито».

### Проверка наличия треков

Альтернатива загрузке: кидаете MP3 в ту же область, расширение сверяет имена с треками на странице и показывает, какие уже есть, а каких нет.

- Сравнение умное: игнорируются `[квадратные]` и `(круглые)` метки в названиях, хвосты `feat. X`, разные тире, регистр.
- Диалог загрузки закрывается сам, сразу после дропа.
- По умолчанию сканируются первые 100 треков (мгновенно). Можно включить «всю страницу» — расширение само проматывает её до конца.
- Результат: панель справа с миникартой и тремя списками — «Отсутствуют», «Уже есть на странице», «Все треки страницы».
- Клик по любой строке или маркеру миникарты — перелетает к нужной аудиозаписи на странице и подсвечивает её.
- У каждой секции своя кнопка копирования: отсутствующие, имеющиеся, все треки в формате `Artist — Title`, отдельно — уникальные имена артистов.

### Поиск дубликатов в плейлисте

Кнопка в попапе плейлиста разворачивает список целиком и помечает:

- пары «исполнитель + название», которые встречаются больше одного раза;
- треки, недоступные по данным VK.

Справа от попапа появляется панель с миникартой позиций и списком найденного. Клик по маркеру/строке прокручивает попап к нужному треку. Список можно скопировать.

### Скачивание

- **Один трек** — иконка скачивания в строке (появляется при наведении). Поддерживает обычные ссылки и HLS-потоки (с расшифровкой AES-128).
- **Отмена** — повторный клик во время скачивания прерывает загрузку.
- **Весь плейлист** — две кнопки в попапе плейлиста: «по одному файлу» или одним ZIP-архивом. Виден прогресс с фазой и счётчиком.

### Настройка раскладки VK

- **Закрепить сайдбар** — левая колонка остаётся на месте при прокрутке страницы, прилипает прямо под верхней панелью.
- **Смещение контента** — слайдер от −400 до +400px сдвигает сайдбар, основной контент и верхнюю панель синхронно влево или вправо. Кнопка «↺» сбрасывает в ноль.

## Установка

1. Скачать или склонировать репозиторий.
2. Открыть `chrome://extensions/` и включить режим разработчика.
3. «Загрузить распакованное расширение» — указать папку проекта.
4. Обновить открытые вкладки vk.com.

## Настройки

Открываются по иконке шестерёнки в шапке диалога загрузки.

| Опция | По умолчанию | Назначение |
|---|---|---|
| Режим работы | Загрузка | Переключатель «Загрузка / Проверка» |
| Сканировать всю страницу | выкл. | В режиме «Проверка» — сканировать всю страницу вместо первых 100 |
| Авто-плейлист | выкл. | Создавать плейлист после очереди загрузки |
| Обложка | — | Картинка 1000×1000 для плейлиста |
| Обложка из ID3 | выкл. | Использовать встроенную обложку первого трека, если своя не задана |
| Авто-метаданные | выкл. | Дополнять отсутствующие теги из имени файла |
| Закрепить сайдбар | выкл. | Левая колонка следует за прокруткой |
| Смещение контента | 0 | Горизонтальный сдвиг макета в пикселях |

## Совместимость

Chromium-браузеры (Chrome, Edge и совместимые), десктопная версия vk.com. Мобильная версия `m.vk.com` не поддерживается. Внешних зависимостей нет.
