# VK Multi Audio Upload

Расширение для Chromium-браузеров, расширяющее работу с музыкой на vk.com: массовая загрузка MP3, автоматическое создание плейлиста, скачивание треков и плейлистов, поиск дубликатов и недоступных записей.

## Возможности

**Загрузка.** Перетаскивание группы MP3 в нативный диалог «Добавить аудиозапись»: файлы выстраиваются в очередь и загружаются по одному. Состояние каждого трека отображается отдельно, упавшие можно перезапустить. ID3v2-теги читаются в браузере; при необходимости отсутствующие исполнитель и название извлекаются из имени файла по шаблону `Исполнитель - Название.mp3` и записываются в файл перед загрузкой.

**Авто-плейлист.** По завершении очереди расширение может само открыть диалог создания плейлиста, заполнить название и описание из тегов альбома и установить обложку. Источник обложки — файл из настроек или встроенное изображение APIC из первого трека.

**Анализ плейлиста.** Кнопка в шапке попапа плейлиста разворачивает список целиком (включая виртуализированные строки) и помечает дубликаты по паре «исполнитель + название», а также треки, недоступные по данным VK. Справа от попапа появляется панель с миникартой позиций и списком недоступных записей; клик по маркеру прокручивает попап к нужной строке.

**Скачивание.** Отдельные треки — по иконке в строке. Целый плейлист — отдельными файлами или одним архивом. Прямые ссылки получаются через `reload_audio`, HLS-потоки — посегментно с расшифровкой AES-128 и извлечением аудио из контейнера MPEG-TS.

## Установка

1. Скачать или склонировать репозиторий.
2. Открыть `chrome://extensions/` и включить режим разработчика.
3. «Загрузить распакованное расширение» — указать папку проекта.
4. Обновить открытые вкладки vk.com.

## Настройки

Открываются по иконке шестерёнки в шапке диалога загрузки.

| Опция | По умолчанию | Назначение |
|---|---|---|
| Авто-плейлист | выкл. | Создаёт плейлист после завершения очереди |
| Обложка | — | Картинка 1000×1000, используется как обложка плейлиста |
| Обложка из ID3 | выкл. | Использовать APIC из первого трека, если своя не задана |
| Авто-метаданные | выкл. | Дополнять отсутствующие теги из имени файла |

Настройки хранятся в `localStorage` под ключом `vmu_settings_v2`.

## Структура

```
manifest.json        MV3-манифест
background.js        service worker, chrome.downloads
injector_early.js    перехват drop на document_start
injected.js          page-context: VK API, React fiber, HLS, ZIP, ID3
content.js           isolated world: UI, очередь, настройки
style.css            стили
```

`content.js` и `injected.js` обмениваются сообщениями через `window.postMessage`; всё, что требует доступа к `window.vk` и React fiber, выполняется в `injected.js`.

## Совместимость

Chromium-браузеры (Chrome, Edge и совместимые), десктопная версия vk.com. Мобильная версия `m.vk.com` не поддерживается. Manifest V3, единственный host-permission — `https://vk.com/*`, внешние зависимости отсутствуют.
